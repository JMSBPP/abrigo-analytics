Working external brand name: **Abrigo** (ah-BREE-go).

Supersedes "ThetaSwap" for all pitch, hackathon, marketing, and user-surface copy. The repository, codebase, and internal engineering documents still use "ThetaSwap" / "thetaSwap-core-dev" until a rename is scheduled \u2014 that is deferred, not urgent.

**Why Abrigo:**
- Spanish for *shelter / coat / warmth*. The meaning IS the product: covering a household from macro weather.
- Two syllables, pronounceable in EN / PT / ES; reads naturally for the Colombian pilot and extends globally to the underserved-FX mission.
- Warm, household-felt, not institutional-jargon \u2014 matches the dual-narrative hero user (household + LP).
- No crypto baggage, no Greek-letter / swap / vault / pool / yield leakage.
- Fits the positioning principles (mainstream, painkiller, crypto abstracted away).

**Prior candidate considered and dropped:** "Reserva" was selected briefly, then reverted. Reserva is too commercially crowded (Brazilian fashion brand, wine grading, hospitality) and lacked the warmth that anchors the household-protection narrative.

**Open risk:** verify before any public launch:
- Domain availability (abrigo.app, abrigo.money, abrigo.finance, abrigo.xyz, getabrigo.com, etc.).
- Trademark clearance in pilot + target jurisdictions.
- A US-based SaaS banking vendor called "Abrigo" exists (bank compliance / risk software); class and geography overlap needs review.
- Qualifier fallbacks if blocked: Abrigo Money, Abrigo Protect, Abrigo.app, Abrigo Wealth, GetAbrigo.

**How to apply:** every pitch artifact, one-pager, application answer, slide, tagline, and demo script uses "Abrigo." If a qualifier becomes required post-clearance, the branding agent updates one config entry and regenerates artifacts.
